Introduction

The Tagger Eclipse Plug-in adds tagging functionality to Eclipse such that resources may be tagged similar to the content tagging in Del.icio.us and Gmail. The tagged resources may then be managed, searched, filtered, or otherwise operated on by their tag associations.

Added Functionality

This plug-in is still in the early development stages; please refer to the requirements/specification documentation in the Documentation section of the web site for more details about the added features.
