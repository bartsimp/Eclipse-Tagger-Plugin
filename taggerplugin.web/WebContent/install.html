Requirements

In order to install and use this plug-in your environment must meet the following minimum requirements:

  • Java 5 (makes use of new features so it will not work on older JVMs)
  • Eclipse 3.2 (may work on later versions, but has not been tested).

Installation

Plug-in Jar

Simply download the plugin jar and drop it into your Eclipse plugins directory and restart Eclipse.

Update Site

Add the update site (http://taggerplugin.sourceforge.net/update) to your update configuration and then get the most current version of the plug-in.
